🇮🇳 उपयोग गाइड (Usage Guide in Hindi)

Property Sale History Analyzer ek complete tool hai jisse aap apni real estate properties ki खरीद-बिक्री history track kar sakte hain, multiple properties compare kar sakte hain, aur ROI (Return on Investment) ka detailed analysis paa sakte hain.

1️⃣ Property Add Karna:

  1. Manual Entry: "Add Property" tab mein property ka naam, location, type, purchase date/price aur (agar bik gayi hai) sale date/price bharein.
  2. Additional Costs mein registration, stamp duty, brokerage jaisi extra lagat dalein — isse accurate profit calculate hoga.
  3. Agar property abhi bhi aapke paas hai (becha nahi), to Sale Date/Price khali rakh sakte hain — tool use "Currently Holding" mark karega.
  4. CSV Upload: Bulk data ke liye "CSV Upload" mode select karein, template download karein, Excel mein bharein aur upload kar dein.

2️⃣ Sale History Dekhna:

  • "Sale History" tab mein saari properties ki table dikhegi — buy/sell price, profit/loss, aur ROI % ke saath.
  • Search box se kisi specific property ya location ko dhoondh sakte hain.
  • Filter se "Sold Only" ya "Currently Holding" properties dekh sakte hain.
  • Sort dropdown se newest, oldest, ya highest/lowest ROI ke hisaab se arrange karein.

3️⃣ Properties Compare Karna:

  • "Compare" tab mein chips par click karke 6 tak properties select karein.
  • Purchase vs Sale price, ROI %, aur Price per Sq.Ft ke charts automatically ban jayenge.

4️⃣ ROI Dashboard:

  • Total investment, total returns, overall profit/loss aur average ROI ek nazar mein dikhega.
  • Portfolio Value Timeline se pata chalega ki samay ke saath aapka investment kaise grow hua.
  • Annualized Returns chart se pata chalega ki kaun si property ne sabse acha CAGR diya.

5️⃣ Data Export & Safety:

  • "Export CSV" button se apna pura data CSV file mein download kar sakte hain — backup ke liye useful.
  • Ye tool 100% client-side hai — koi bhi data server par nahi jata, sab kuch sirf aapke browser mein (local storage) save hota hai.
  • Browser cache/history clear karne se data delete ho sakta hai — important data CSV export karke रखें।

⚠️ Note: Yeh tool sirf calculation aur tracking ke liye hai, investment advice nahi deta. Koi bhi bada financial decision lene se pehle apne CA ya financial advisor se salah lein.
